# Build mode — helping the user start their Top 100

The user has 0-4 names. Goal: end this session with 1-5 new names added.

## Flow

### 1. Tell the user where they are
"You have <N> names so far. Let's add 1-5 more. I'll ask 3 questions to help you find them."

### 2. The 3 questions

**Q1. Who has audiences that match yours?**
"Think about: podcasters, newsletter writers, list owners. Whose audience would buy your offer if they trusted you? Name 1-3 people."

After each name, save row to `dream-100.md` with category "list_owner".

**Q2. Who do you already admire in your space?**
"Think about: people you follow, courses you bought, communities you're in. The people you'd want to be associated with. Name 1-3 people."

Save with category "admired".

**Q3. Who's adjacent — serving your buyer with a different thing?**
"Think about: people who serve your same audience but sell something different — complementary, not competing. Name 1-3 people."

Save with category "adjacent".

### 3. Load `business-brain.md` ICP definition
If the user's ICP is in business-brain, suggest 2-3 names that match. Frame as: "These came to mind because they fit your ICP — but only add if YOU recognize the fit."

### 4. Auto-switch at 5+ names
If the list crosses 5 names this session, surface:
"Nice — you're at 5+ names. Next time you run /dream-100, it'll be in Operate mode. You'll start being prompted to touch them (with generosity first)."

### 5. Self-improvement close + Next Move

```
⚡ NEXT MOVE: Follow <name added 1> on <platform> today.
   Why: First step in the 3-6 month rapport runway — start consuming their content.
```
