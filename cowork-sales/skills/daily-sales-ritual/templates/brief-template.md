# Daily brief template

The standard 3-column brief structure with personalization-aware outbound prompt.

```
═══════════════════════════════════════════
  YOUR DAILY SALES RITUAL — <YYYY-MM-DD>
═══════════════════════════════════════════

📨 EMAIL PIPELINE (<N> names)
  <for each name: bullet with score + suggested action>
  Format: • Name — <buyer-pyramid bucket signal> → <suggested action>

🎙️ FROM YESTERDAY'S ACTIVITY (<N> names)
  <for each Fathom mention: name + 1-sentence context>
  Format: • Name — <what they said or context>

⏰ OVERDUE FOLLOW-UPS (<N> names)
  <for each overdue: name + days overdue + suggested action>
  Format: • Name — <X> days overdue → <suggested action>

═══════════════════════════════════════════
TODAY'S OUTBOUND PROMPT (rotating from your tactics)
  → <channel name>: <personalized prompt from channel file>
═══════════════════════════════════════════

<NEXT MOVE BLOCK from templates/next-move-block.md>
```

## Special states

### Missed 2-3 days: insert at top, before the 3 columns:
```
⚠️ 60-SECOND RE-ENGAGEMENT
  You missed <N> days. Take a breath. Today's brief picks up where
  you left off. Two things to remember before you start:
  1. <re-orient: what's still warm?>
  2. <re-orient: what's the one move that matters?>
```

### Missed 7+ days: replace full brief with soft restart:
```
═══════════════════════════════════════════
  SOFT RESTART — <YYYY-MM-DD>
═══════════════════════════════════════════

Big gap since your last ritual. Streak reset. That's fine —
the system is designed for 365 days, not 30.

Today, we're focusing on 1-2 reach-outs only:
  • <The single highest-priority overdue warm follow-up>
  • <The single highest-quality recent inbound>

Skip the full brief. Just do these two.

<NEXT MOVE BLOCK — focused on ONE of the two>
```

### No email pipeline yet (Gmail just connected):
```
📨 EMAIL PIPELINE
  No email pipeline yet — Gmail just connected. Your first real
  brief is tomorrow.
```
